# Vinternet Music Vault Desktop (Electron Application)

## Setup & Usage On Local Node Environment

### Requirements

- [NodeJS][nodejs] 12.0.0 and above
- [Yarn][yarn] (you can still use npm if you wish)
- [Electron][electron]
